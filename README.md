# 12 Pions

Jeu des 12 pions : plateau 5×5, jouable en ligne et contre l’ordinateur + classement des joueurs.

## Lancer en local

```bash
npm install
npm run dev
```

- Site (Vite) : http://localhost:5173  
- API / WebSocket : http://localhost:3001  

## Production

```bash
npm run build
npm start
```
